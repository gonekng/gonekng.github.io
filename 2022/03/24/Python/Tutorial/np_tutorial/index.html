파이썬 라이브러리 설치

in R (코드에서 실행)

  • install.package(“패키지명”)
  • library(패키지명)

in Python (터미널에서 실행)

  • 방법1. conda 설치 (주 사용목적: 데이터 과학)

  • 방법2. pip 설치 (개발, 데이터 과학, 그 외)
    • 아나콘다 없이도 python만 설치하면 됨
