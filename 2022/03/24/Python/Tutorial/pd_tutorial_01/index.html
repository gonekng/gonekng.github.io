iloc vs loc

  • iloc : index 기반, 속도↑, 대용량 데이터에 적합
    • syntax : df.iloc[row_index, column_index]
  • loc : label or boolean(조건식) 기반, 가독성↑
    • syntax : df.loc[row_label, column_label]
